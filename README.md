# SAE Web Development 2016

## get repository
```sh
git clone git@github.com:gruppjo/sae-webdev.git
```

## run slideshow

```
npm install -g reveal-md
reveal-md --theme solarized
```

## run examples

```sh
# in directory of lesson
npm install
bower install
gulp serve
```
For further instructions (dependencies, etc.) see `../lesson1.md`
